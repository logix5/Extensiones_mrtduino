# Extension_TFT_ST7735_mrtduino
Extensión para usar el TFT de color. 

Conexión

VCC  ---> 5V

GND  ---> Masa (-)

CS, RESET, DC(A0), se eligen al usar la extensión

Led  ---> 3,3 Voltios

SDA  ---> MOSI de la placa MRTDuino

SCK  ---> SCK de la placa MRTDuino
