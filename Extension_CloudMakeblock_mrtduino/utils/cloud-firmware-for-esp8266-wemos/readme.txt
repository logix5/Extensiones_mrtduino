Firmware para conectar un robot de Makeblock al cloud (http://cloud.makeblock.es) usando un módulo wifi ESP8266 Wemos ( o similar )

Version: 3

Por: Jose Carlos ( jose@makeblock.es)



1: Conectar el módulo Wifi al PC usando un cable micro-usb (en caso de wemos) o un FTDI si no tiene conector USB
2: Ejecutar flash_firmware.cmd   y esperar a que finalice
